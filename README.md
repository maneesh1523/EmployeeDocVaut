# EmployeeDocVaut

A document vault app with RBAC (Role-Based Access Control) using a FastAPI/Python backend and Vite/React frontend.

## Run locally

1. Backend: `python -m venv venv && venv\Scripts\activate && pip install -r requirements.txt && uvicorn backend.main:app --reload`
2. Frontend: `cd frontend && npm install && npm run dev`
